Comp Org'n & Assem Lang Prog

CDA 3104

Course Description

Covers the basic concepts of computer organization using a computer platform or a simulator and the corresponding assembly language. Topics include memory models, register and CPU structures, basic assembly instructions, procedures, stack frames, and interrupt handling.
